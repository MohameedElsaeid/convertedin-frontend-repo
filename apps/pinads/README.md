### - Project commands:

#### Serve

```
npm run pinads:serve
```

#### Build stage

```
npm run pinads:build:staging
```

#### Build production

```
npm run pinads:build:prod
```

#### Build analyze

```
npm run pinads:build:analyze
```

#### Analyze build

```
npm run pinads:analyze
```
