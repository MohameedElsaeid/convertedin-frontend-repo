### - Project commands:

#### Serve

```
npm run flyerz:serve
```

#### Build stage

```
npm run flyerz:build:staging
```

#### Build production

```
npm run flyerz:build:prod
```

#### Build analyze

```
npm run flyerz:build:analyze
```

#### Analyze build

```
npm run flyerz:analyze
```
